Project 1: Real-Time Stock Tracker

Overview / Summary

This app tracks stock prices and related financial news in real time. It’s built with plain HTML, CSS, and JavaScript for simplicity and speed. Users enter a stock symbol to get the latest data and news.

Process & Methodology

  • Used semantic HTML to create a clear and organized layout.
  • Built modular JavaScript functions to handle API requests and update the UI.
  • Applied CSS Flexbox for a responsive design that works on all screen sizes.
  • Iterative testing and debugging using Git and browser dev tools.

Key Features & Deliverables

  • 🔍 Search bar for stock symbols
  • 📈 Real-time price data
  • 📰 Shows company-specific news feed
  • 📱 Fully responsive
  • ❌ Handles invalid inputs and API errors

Results & Achievements

  • Data loads in under 2 seconds
  • Zero frameworks used
  • Modular and readable code designed for easy scaling

Learnings & Growth

  • Mastered fetch and async/await
  • Enhanced skills in front-end structure and responsive design
  • Learned clean code practices without relying on frameworks
