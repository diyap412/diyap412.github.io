Project 1: Real-Time Stock Tracker

Overview/Summary

This is a lightweight stock tracking app built using vanilla HTML, CSS, and JavaScript. It shows real-time stock data and financial news based on user input.

Process & Methodology

  • Structured layout using semantic HTML.
  • Modular JavaScript for API calls and UI rendering.
  • CSS Flexbox for responsive layout.
  • Iterative testing and debugging using Git and browser dev tools.
